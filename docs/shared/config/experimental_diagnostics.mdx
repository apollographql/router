### `experimental_diagnostics`

Configuration for the experimental diagnostics plugin, which provides a dashboard and API for inspecting router configuration, system and memory information, and exporting diagnostic packages. See [Diagnostics (experimental)](/graphos/routing/self-hosted/diagnostics) for full documentation.

```yaml title="experimental_diagnostics"
experimental_diagnostics:
  enabled: false
  listen: 127.0.0.1:8089
  output_directory: /tmp/router-diagnostics
```
