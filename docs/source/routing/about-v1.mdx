GraphOS Router v{products.router.version("lts_1_latest").version} is the Apollo runtime platform.

## Release status of router 1.x

GraphOS Router v1.x is in **Maintenance** with latest version {products.router.version("lts_1_latest").version}.

| Current Status | Release Date | Latest Minor | Active Date    | Maintenance Date | End of Life Date |
| -------------- | ------------ | ------------ | -------------- | ---------------- | ---------------- |
| Maintenance    | August 2022  | v{products.router.version("lts_1_latest").version}  | September 2022 | February 2025     | Planned February 2026       |

As a [maintenance](/graphos/reference/router-release-lifecycle#maintenance) release:

- Only the latest minor version ({products.router.version("lts_1_latest").version}) receives security patches and critical bug fixes. 
    - If a bug fix requires incrementing the minor version to conform to semantic versioning, subsequent fixes and patches would be applied to this latest incremented minor version.

- No features are added or are under development.

- [Support](https://www.apollographql.com/enterprise/support) is provided via paid support plans until end-of-life (planned for February 2026). 

## What's new in router 1.x

Refer to the [v{products.router.version("lts_1_latest").version} changelog](https://github.com/apollographql/router/releases/tag/v{products.router.version("lts_1_latest").version}) for the latest updates to router 1.x. 
  
## Upgrading to router 1.x

If you are upgrading to v{products.router.version("lts_1_latest").version} requires the following:

- **Upgrade your supergraphs to Federation v2**. 

   Federation v2.10 will be the last supported federation version for router 1.x LTS.

## Upgrading to latest router

The latest router version is v{products.router.version("lts_1_latest").version}. To upgrade to v{products.router.version("lts_1_latest").version}, follow the [upgrade guide from router 1.x to 2.x](https://www.apollographql.com/docs/graphos/reference/upgrade/from-router-v1).
