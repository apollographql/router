| Attribute                | Default | Description                                                                              |
|--------------------------|---------|------------------------------------------------------------------------------------------|
| `scheduled_delay`        | 5s      | The delay from receiving the first span to the batch being sent.                         |
| `max_concurrent_exports` | 1       | The maximum number of overlapping export requests.                                       |
| `max_export_batch_size`  | 512     | The number of spans to include in a batch. Your ingest may have max message size limits. |
| `max_export_timeout`     | 30s     | The timeout for sending spans before dropping the data.                                  |
| `max_queue_size`         | 5s      | The maximum number of spans to be buffered before dropping span data.                    |
