### `cors`

Cross origin request configuration.



```yaml title="cors"
cors:
  allow_any_origin: false
  allow_credentials: false
  allow_headers: []
  expose_headers: null
  match_origins: null
  max_age: null
  methods:
  - GET
  - POST
  - OPTIONS
  origins:
  - https://studio.apollographql.com
```
