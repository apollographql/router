### Add documentation for `experimental_hoist_orphan_errors` configuration

Added the `experimental_hoist_orphan_errors` configuration option to the YAML configuration reference. The new documentation covers the feature's purpose (reducing multiplicative error propagation from entity resolution), per-subgraph and global enablement examples, the spec compliance caveat, and the known limitation that error counts are reduced but not hard-capped.

By [@andywgarcia](https://github.com/andywgarcia) in https://github.com/apollographql/router/pull/8999
